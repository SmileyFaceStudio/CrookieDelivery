# PirateKitchen
Crookie Delivery Web Application built in AngularJS

### Installation
Clone application
```sh
npm install -g http-server bower
cd <repo name>
bower install
http-server -c-1
```

### Information
1. Takes your order
2. Enter your information
3. Review your order and information
4. Sends it to Google Sheets API
